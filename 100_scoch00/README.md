# スコッチヨーク機構

## 作りたいもの

- モータ（ステッピングorDC）と連動させ、直線運動をさせたい
- まずは小型のスコッチヨーク機構をつくる

## 概要

　スコッチ･ヨーク機構は，出力軸に偏心して取り付けられたクランクピン軸受が長円形の溝の中を転がることにより，ヨークを往復動させる機構です。
